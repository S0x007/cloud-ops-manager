### 4. 表格行交互
- **单击行** → 右侧面板显示详情（不分页跳转）
- **双击行** → 全屏详情页
- **右键行** → 上下文菜单
- **复选框 + 工具栏** → 批量操作

## 四、状态设计模式

每个资源页面必须覆盖以下 5 种状态：

| 状态 | 触发条件 | UI 表现 |
|------|---------|--------|
| **加载中** | 首次数据获取 | Ant Skeleton（骨架屏，不是 Spin） |
| **空数据** | 当前 Region 无资源 | Empty + 描述 + CTA 按钮（如"创建实例"） |
| **错误** | API 失败 | Alert + 错误详情 + 重试 + 权限提示 |
| **权限不足** | IAM 拒绝 | 特定提示：缺少哪个权限 + 如何添加 |
| **正常** | 数据加载成功 | 标准表格/卡片 |

## 五、视觉设计规范

### 色板
```
主色:    #1677ff (Ant Design Blue)

警告:    #faad14 (Gold)     → stopping / pending / in-use
错误:    #ff4d4f (Red)      → stopped / terminated / error

```

## 六、快捷键体系

| 快捷键 | 操作 |
|--------|------|
| `Cmd+K` | 全局命令面板 |
| `Cmd+R` | 刷新当前视图 |
| `Cmd+1-5` | 切换侧边栏菜单（Dashboard/EC2/S3/ECS/设置） |
| `Cmd+[` | 返回上一页 |
| `Cmd+]` | 前进下一页 |
| `Esc` | 关闭弹窗/抽屉/命令面板 |
| `Enter` | 确认操作 |
| `Space` | 勾选当前行（表格内） |

## 九、响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|---------|
| 大屏 | ≥1400px | 侧边栏展开 + 分栏布局 |
| 中屏 | 1024-1399px | 侧边栏折叠 + 单列布局 |
| 小屏 | <1024px | 侧边栏隐藏 + 汉堡菜单 |
