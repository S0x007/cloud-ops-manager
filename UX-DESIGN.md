# Cloud Ops Manager — UX 设计规范

> 2026-06 更新：应用已支持 **AWS / 华为云** 双 Provider。侧栏菜单与 Region 列表随顶栏云厂商切换而变化。下文「AWS 视图」与「华为云视图」分开描述。

## 一、信息架构

### 1.1 全局

```
Home (Dashboard)          ← 统计卡片随 Provider 切换
Settings / 凭证管理        ← 按 provider 字段区分 AK/SK
```

顶栏：**折叠 · Cmd+K · 云厂商▾ · 凭证▾ · 区域▾ · 验证状态 · 语言 · 主题**

### 1.2 AWS 视图

```
Dashboard
├── EC2
│   ├── 实例列表
│   └── 实例详情 :id（概览/监控/Run Command/控制台/卷/标签/网络）
├── S3
│   ├── 桶列表（创建/清空/删除）
│   ├── 桶详情 :bucket（7 Tab）
│   └── 对象浏览 :bucket/*
├── ECS
│   ├── 集群列表
│   └── 集群详情 :clusterName（服务 Tab · 任务 Tab · 任务 Drawer）
├── 存储：EBS 卷 · 快照
├── 网络：安全组 · 弹性 IP · VPC 总览 · AMI
└── Terminal：SSM · 端口转发
```

### 1.3 华为云视图

```
Dashboard
├── ECS（列表 · 创建 · 详情：VNC/COC/卷/EIP）
├── OBS（桶+对象单页浏览器）
├── EVS / RDS / IMS（列表；RDS 有详情 Tab）
├── 网络：VPC 总览 · 弹性公网 IP
└── Settings
```

## 二、导航设计

### 侧边栏（动态）

- 品牌区文案随 Provider 变化（AWS / Huawei Cloud）
- 菜单项来自 `ALL_PROVIDERS[provider].menus`（`src/renderer/src/providers/`）
- 切换云厂商时：清空资源 Store、过滤凭证、重置 Region 列表

**AWS 侧栏分组示例：** 计算(EC2/ECS) · 存储(S3/EBS/快照) · 网络(安全组/EIP/密钥/VPC) · 其他(AMI) · 设置

**华为云侧栏分组示例：** 计算(ECS) · 存储(OBS/EVS) · 网络(VPC/EIP) · 数据库(RDS) · 其他(IMS) · 设置

### 顶部栏
```
┌──────────────────────────────────────────────────────────────────────┐
│ 折叠  Cmd+K  │  [AWS ▾]  凭证▾  区域▾(含计数)  验证  │  语言  主题  │
└──────────────────────────────────────────────────────────────────────┘
```

### 面包屑
```
S3 > my-bucket > logs/2024/     ← 支持逐级返回
```

## 三、核心交互模式

### 1. 全局命令面板 (Cmd+K)

- **AWS：** 搜索 EC2 实例（ID/名称/IP）、S3 桶；快捷入口 SSM/快照/S3/设置
- **华为云：** 搜索 ECS、OBS 桶、RDS；快捷入口 ECS/OBS/RDS/设置
- 底部 Tag 显示当前 Provider

### 2. 右键上下文菜单（EC2 实例为例）
```
┌────────────────┐
│ 📋 复制实例ID   │
│────────────────│
│ ▶ 启动          │
│ ⏹ 停止          │
│ 🔄 重启          │
│ ⛔ 终止（需输入实例 ID 确认）  │
│────────────────│
│ 🔗 SSM 连接     │
│ 📋 控制台输出   │
│ 🔑 获取密码     │
│────────────────│
│ 💾 创建 AMI     │
│ 📸 创建快照     │
│ 🏷 编辑标签     │
└────────────────┘
```

### 3. 分栏布局（列表+详情）
```
┌────────────────────┬──────────────────────┐
│ 实例列表               │ 实例详情               │
│ ┌────────────────┐   │ i-0a1b2c3d4e5f      │
│ │ □ i-xxx  running│   │                      │
│ │ □ i-yyy  stopped│   │ [概览] [监控] [安全组]  │
│ │ □ i-zzz  running│   │                      │
│ │                │   │ ID: i-0a1b2c3d4e5f   │
│ │                │   │ 类型: t3.medium       │
│ └────────────────┘   │ 状态: 🟢 running      │
│                      │ ...                  │
└──────────────────────┴──────────────────────┘
```

### 4. 表格行交互
- **单击行** → 右侧面板显示详情（不分页跳转）
- **双击行** → 全屏详情页
- **右键行** → 上下文菜单
- **复选框 + 工具栏** → 批量操作

## 四、状态设计模式

每个资源页面必须覆盖以下 5 种状态：

| 状态 | 触发条件 | UI 表现 |
|------|---------|--------|
| **加载中** | 首次数据获取 | Ant Skeleton（骨架屏，不是 Spin） |
| **空数据** | 当前 Region 无资源 | Empty + 描述 + CTA 按钮（如"创建实例"） |
| **错误** | API 失败 | Alert + 错误详情 + 重试 + 权限提示 |
| **权限不足** | IAM 拒绝 | 特定提示：缺少哪个权限 + 如何添加 |
| **正常** | 数据加载成功 | 标准表格/卡片 |

## 五、视觉设计规范

### 色板
```
主色:    #1677ff (Ant Design Blue)
成功:    #52c41a (Green)    → running / available / enabled
警告:    #faad14 (Gold)     → stopping / pending / in-use
错误:    #ff4d4f (Red)      → stopped / terminated / error
信息:    #1677ff (Blue)     → 标签 / 信息
```

### 状态徽章
```
EC2:  🟢 running  🟡 pending  🔴 stopped  ⚫ terminated
卷:   🟢 in-use   🔵 available  🔴 error
快照: 🟢 completed  🟡 pending  🔴 error
安全组: 默认无状态徽章
```

```
弹性 IP:     GlobalOutlined

```

## 六、快捷键体系

| 快捷键 | 操作 |
|--------|------|
| `Cmd+K` | 全局命令面板 |
| `Cmd+R` | 刷新当前视图 |
| `Cmd+1-5` | 切换侧边栏菜单（Dashboard/EC2/S3/ECS/设置） |
| `Cmd+[` | 返回上一页 |
| `Cmd+]` | 前进下一页 |
| `Esc` | 关闭弹窗/抽屉/命令面板 |
| `Enter` | 确认操作 |
| `Space` | 勾选当前行（表格内） |

## 九、响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|---------|
| 大屏 | ≥1400px | 侧边栏展开 + 分栏布局 |
| 中屏 | 1024-1399px | 侧边栏折叠 + 单列布局 |
| 小屏 | <1024px | 侧边栏隐藏 + 汉堡菜单 |
