# Cloud Ops Manager v2.0.1

体验抛光版本 — S3 预签名链接、华为 RDS 规格可读、OBS 目录统计、应用内自动更新。

---

## 下载

| 平台 | 文件 |
|------|------|
| **Windows** | `Cloud.Ops.Manager.Setup.2.0.1.exe` |
| **macOS** | `Cloud.Ops.Manager-2.0.1-arm64.dmg`（Apple Silicon） |
| **Linux** | `Cloud.Ops.Manager-2.0.1.AppImage`（x64） |

另附 `latest*.yml` 与 `*-arm64-mac.zip` 供应用内更新。

---

## 更新说明

### v2.0.1

- **S3**：对象操作栏「预签名链接」，可选 15 分钟 / 1 小时 / 24 小时有效期  
- **华为 RDS**：规格列显示 `2 vCPU / 4 GB`，隐藏原始 flavor 名  
- **华为 OBS**：二级目录聚合显示大小与最近修改时间（超大目录显示 `~` 表示部分统计）  
- **自动更新**：设置 → 软件更新，对接 GitHub Releases  

### v2.0.0

- 首发多云版：AWS + 华为云、凭证加密、双语界面
